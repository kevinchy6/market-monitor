# Market Monitor

Real-time equity market dashboard tracking ETFs across sectors, indices, and global markets.

**Columns:** Market, Symbol, Price, WTD, MTD, YTD  
**Features:** Click column headers to sort, auto-refresh every 5 minutes

## Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set Source to **Deploy from a branch**
4. Select **main** branch, **/ (root)** folder
5. Click Save — site will be live at `https://<username>.github.io/<repo-name>/`

## Data

`data.json` contains cached OHLCV data. To update, replace with fresh data from Yahoo Finance.
